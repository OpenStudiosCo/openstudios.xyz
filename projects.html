Kettlefish

Templates and tools for efficiently building websites and software with components tested through Kamigen and Manifold

An opinionated boilerplate to create static websites. Out of the box Kettlefish comes with a default homepage built using a Semantic UI layout example, allowing anyone who knows what these tools are to hit the ground running.

Manifold

Our flagship software application, aimed to advance understanding and capabilities with digital media production and publishing

A hybrid 2D/3D design tool, this is an experimental program to address the need for fast and efficient modelling for Three.JS scenes. The 2D vector component can also be used for general illustration and design needs.
